# Achievements
Every set includes achievements that are scripted in a certain way. In the README file I will mainly justify my reasoning for the naming. For code logic, you should refer to the code notes and the script itself.

## Naming 
Naming the achievements is done according to the [Achievement Design Docs](https://docs.retroachievements.org/developer-docs/achievement-design.html).

The achievements are devided into several parts that also exclude sosme modes. I will justify those decisions here.
The first section is the Puzzle Mode, the second part is the Player vs Computer Mode, the third is the Challenge Mode, the fourth part is the Win Contest Mode, the fifth is the Collection Mode.

### Puzzle Mode
Puzzle Mode makes you pop bubbles, just like every other mode, in order to progress you have to finish a set or a long puzzle. The Set of Puzzles is always named a letter, but some Sets are made to play like one longer Stage Which are called Endurance Stages, that reward clearing out the visible playing field. Those endurance Stages are B, D, G, K and M. The other Sets are made up of 5 Stages that only differ between Modes and Another World.
Every last Letter leads to a new type of Scene, which makes every choice more memorable then just another playthrough. Making an achievement for every letter seems like a fun idea, but it would simply flood the Core Set with a lot of achievements, keep in mind the achievements without including every stage is already 28 Achievements, in that case you would make it roughly 72 Achievements. That seems more fitting for a Subset in my opinion, which is why I kept the overall scope smaller. You are awarded for all the progress, without having to do a Mode again until completion.
I also did not make an achievement for every Character you used to complete the Puzzle Mode for the same reasons, not many will realize that the Puzzle Mode does in fact have a Character Select at the same time you select a Set. This is why you can see which Character is playing in Puzzle Mode.

I chose to make an achievement for every level of the Set group and for every Mode, and every Mode in Another World. Which means that there are roughly four Sets. For every ending of those Sets there is also a Challenge achievement that makes you beat every last Stage in Very Hard. Which means that if you start out in Very Hard, you only need to do one playthrough, which rewards players that are familiar to the game, as to not have them feel as if they are grinding it out. But to also sufficiently rewarding new players, for playing through it the normal way.

In Puzzle Mode, you also have a Practice Mode, I excluded it as it is pretty much just a guide on how to play. I did consider making achievements for simply completing this mode, or doing it in a certain time.
The speedrun statistics, and my own speedrun statistics just did not really give me much of an indication as luck was partly involved. Then just completing it complicated the scoring of all other achievements. Apart from that issue, Practice Mode did not seem memorable, so I decided to leave it out to make the core achievement set overall more enjoyable. The fact that you could in fact measure it very well and finish it in a certain time made it fitting for a Leaderboard. Which means you can choose to grind it out in order to get the fastest time on the Leaderboard. That way grinding with skill is not a requirment for the Achievement Set, but an option.

The naming format is as follows:

`Complete Set {Letters of Sets included} of Puzzle Mode on {DifficultyMode} {in Another World} {at Very Hard Game Level}`

It makes it one flowing line of text that includes all the information one needs to complete the Achievement. I left out the fact that you can't play on Easy game settings or handicap. Since it never said you could change the settings. I only include the Very Hard Game Level, since this makes you go into your settings to change it.
